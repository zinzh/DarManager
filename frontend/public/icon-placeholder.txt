To generate proper PWA icons:

1. Create a 512x512 PNG icon with your logo
2. Use an online tool like https://www.pwabuilder.com/imageGenerator
3. Or use https://realfavicongenerator.net/
4. Save the generated icons in the public/icons folder with these names:
   - icon-72x72.png
   - icon-96x96.png
   - icon-128x128.png
   - icon-144x144.png
   - icon-152x152.png
   - icon-192x192.png
   - icon-384x384.png
   - icon-512x512.png

For now, the app will work with the placeholder icon.
